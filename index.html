Frequently Asked Questions

  • What is ChatGPT?

    ChatGPT is a chatbot powered by artificial intelligence and created by OpenAI. Its ability to understand and generate text and code surpassed anything that had been seen before, leading to its widespread adoption by hundreds of millions of users.

  • What is the difference between GPT-3, GPT-4 and ChatGPT??

    GPT-3 is a state-of-the-art language model that has been trained on a massive corpus of text data using unsupervised learning techniques. It is capable of generating human-like responses to prompts given to it in natural language, and has been used for a wide range of applications, such as language translation, content generation, and chatbot development.

    ChatGPT, on the other hand, is a specific implementation of the GPT architecture that has been fine-tuned for the task of conversational AI. It has been specifically trained to generate responses to natural language inputs from users, and has been used for applications such as customer service chatbots, virtual assistants, and chat-based games.

    In summary, GPT-3 and GPT-4 are general-purpose language models that can be used for a wide range of applications, while ChatGPT is a specific implementation of the GPT architecture that has been optimized for conversational AI.

  • How do I add GPT-3 or GPT-4 to Google Docs??

    The ChatGPT for Docs extension lets you choose the model from OpenAPI settings Menu and you want to use for each GPT function. As a result, you can select a GPT-3 or GPT-4 model if you prefer that over the ChatGPT model.

    To select a GPT-3 or GPT-4 model in Docs, select the model technical name (e.g. “text-davinci-003” or “gpt-4”) in the models dropdown in the OpenAI Settings.
